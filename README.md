# Mental Health Support Chatbot (LLM)

## Overview
This project demonstrates a domain-specific chatbot designed to provide empathetic and supportive responses to users experiencing stress, fatigue, anxiety, and low motivation.

## Approach
- Generated a synthetic dataset of emotional conversations
- Used Smolify to perform distillation-based fine-tuning
- Deployed the model using Hugging Face Transformers

## Model
Hugging Face Model:
https://huggingface.co/Noyonika20/noyonika-mental-health-chatbot

## Features
- Emotion-aware responses
- Context-sensitive support
- Lightweight and efficient model

## Example

**Input:**
I feel overwhelmed and tired all the time  

**Output:**
It sounds like you're carrying a lot right now. It's okay to feel this way, and you're not alone.

## Files in this Repo
- Dataset (synthetic conversational data)
- Colab notebook for inference/testing

## Disclaimer
This chatbot is for supportive interaction only and is not a substitute for professional mental health care.
